# Jeu du Morpion en Python

## 🎮 Présentation
Petit projet personnel pour mettre en pratique mes bases en Python. C'est un jeu de Morpion classique qui se joue à deux dans le terminal.

## 🛠️ Compétences travaillées
* Utilisation des listes pour la grille.
* Gestion des boucles `while` et des conditions `if/else`.
* Création de fonctions pour vérifier les gagnants.

## 🚀 Comment tester ?
Il suffit de lancer le fichier `morpion.py` avec Python 3.
